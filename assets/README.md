# Assets Folder

## Required Images

Please add the following images to this folder:

### 1. shaheen-logo.png
- The Shaheen Cricket Club logo (use one of the provided logo images)
- Recommended format: PNG with transparent background
- Recommended size: 500x500px minimum

### 2. profile.png
- Your profile photo (Anil Kakkar)
- Must have background removed (transparent/cutout)
- Recommended format: PNG
- Recommended size: 800x1000px for best quality
- Should be professional and suitable for portfolio

## Instructions

1. Save your club logo as `shaheen-logo.png`
2. Save your profile photo as `profile.png`
3. Place both files in this `/assets/` folder

The website will automatically load these images when you open index.html.

---

**Note:** If you don't have a background-removed profile photo, you can use online tools like:
- remove.bg
- Photoshop
- GIMP (free)
